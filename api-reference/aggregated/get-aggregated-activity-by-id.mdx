# GET `/api/aggregated/{id}`

Fetch a specific aggregated activity by its unique ID.

## Path Parameters

| Name | Type   | Required | Description                  |
|------|--------|----------|------------------------------|
| `id` | `string` | yes        | Aggregated activity ID       |

## Authentication

This endpoint requires a **Bearer Token**.

