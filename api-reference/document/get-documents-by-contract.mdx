# GET `/api/document/contract/{addr}`

Fetch all documents associated with a given contract address.

## Path Parameters

| Name   | Type     | Required | Description         |
|--------|----------|----------|-------------------|
| `addr` | `string` | yes      | Contract address    |

## Authentication

This endpoint requires a **Bearer Token**.

## Example Request

```bash
curl -X GET "https://api.zephyra.dev/api/document/contract/0xcontract1" \
  -H "Authorization: Bearer <token>"
```
