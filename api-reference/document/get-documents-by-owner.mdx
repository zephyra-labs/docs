# GET `/api/document/owner/{owner}`

Fetch all documents for a given owner.

## Path Parameters

| Name    | Type     | Required | Description            |
|---------|----------|----------|------------------------|
| `owner` | `string` | yes      | Wallet address of owner |

## Authentication

This endpoint requires a **Bearer Token**.

## Example Request

```bash
curl -X GET "https://api.zephyra.dev/api/document/owner/0x123abc" \
  -H "Authorization: Bearer <token>"
```
