# GET `/api/document/{tokenId}`

Fetch detailed information about a document using its unique token ID.

## Path Parameters

| Name | Type | Required | Description |
|------|------|-----------|--------------|
| `tokenId` | `integer` | yes | Document token ID |

## Authentication

This endpoint requires a **Bearer Token**.

## Example Request

```bash
curl -X GET "https://api.zephyra.dev/api/document/123" \
  -H "Authorization: Bearer <token>"
```
