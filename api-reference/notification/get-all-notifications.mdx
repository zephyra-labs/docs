# GET `/api/notification`

Get all notifications. No parameters required.

## Parameters

None.

## Responses

### 200 — OK

Returns a list of notifications for the user.

```json
[
  {
    "id": "notif-uuid-123",
    "userId": "user-uuid-456",
    "executorId": "user-uuid-789",
    "type": "system",
    "title": "New Trade Completed",
    "message": "Your trade #123 has been completed successfully.",
    "read": false,
    "createdAt": 1699286400000,
    "updatedAt": 1699372800000,
    "extraData": {
      "additionalProp1": {}
    }
  }
]
```

### 401 — Unauthorized

Returned if the user is not authenticated.

```json
{
  "error": "Unauthorized"
}
```

## Authentication

This endpoint requires a **Bearer Token** for authentication.

## Example Request

```bash
curl -X GET "https://api.zephyra.dev/api/notification" \
  -H "Authorization: Bearer <token>" \
  -H "Accept: application/json"
```
