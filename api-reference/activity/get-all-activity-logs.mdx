# GET `/api/activity`

Fetch all activity logs recorded in the system.  
Supports optional query filters for account, transaction hash, and pagination.

## Query Parameters

| Name | Type | Required | Description |
|------|------|-----------|-------------|
| `account` | `string` | No | Filter results by account address |
| `txHash` | `string` | No | Filter results by transaction hash |
| `limit` | `integer` | No | Maximum number of results to return |
| `startAfterTimestamp` | `integer` | No | Return results after this timestamp (for pagination) |

## Authentication

This endpoint requires authentication using a **Bearer Token**.

## Example Request

```bash
curl -X GET "https://api.zephyra.dev/api/activity?account=0xabc123&limit=10" \
  -H "Authorization: Bearer <token>"
```
