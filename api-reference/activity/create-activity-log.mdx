# POST `/api/activity`

Create a new activity log entry.  
Used to record user or system activities related to blockchain events, contract interactions, or general application usage.

## Request Body

| Name              | Type   | Required | Description                                                       |
| ----------------- | ------ | -------- | ----------------------------------------------------------------- |
| `timestamp`       | `number` | Yes      | Unix timestamp when the activity occurred                         |
| `type`            | `string` | Yes      | Type of activity (`onChain` or `offChain`)                        |
| `action`          | `string` | Yes      | Action performed (e.g. `"CREATE_CONTRACT"`, `"UPDATE_PROFILE"`)   |
| `account`         | `string` | Yes      | Wallet address of the user performing the action                  |
| `txHash`          | `string` | No       | Related transaction hash if on-chain                              |
| `contractAddress` | `string` | No       | Smart contract address involved                                   |
| `extra`           | `object` | No       | Additional contextual data                                        |
| `onChainInfo`     | `object` | No       | Optional blockchain metadata (status, blockNumber, confirmations) |

### Example Request Body

```json
{
  "timestamp": 1731248583,
  "type": "onChain",
  "action": "CREATE_CONTRACT",
  "account": "0xabc123...",
  "txHash": "0xdef456...",
  "contractAddress": "0x789ghi...",
  "extra": {
    "network": "Polygon"
  }
}
```

This endpoint requires authentication using a **Bearer Token**.
