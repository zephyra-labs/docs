# GET `/api/activity/{account}`

Fetch all activity logs for a given wallet address.  
Supports pagination through query parameters.

## Path Parameters

| Name | Type | Required | Description |
|------|------|-----------|-------------|
| `account` | `string` | Yes | Wallet address used to filter activity logs |

## Query Parameters

| Name | Type | Required | Description |
|------|------|-----------|-------------|
| `limit` | `integer` | No | Maximum number of results to return |
| `startAfterTimestamp` | `integer` | No | Return results after this timestamp (for pagination) |

## Authentication

This endpoint requires authentication using a **Bearer Token**.

## Example Request

```bash
curl -X GET "https://api.zephyra.dev/api/activity/0xabc123?limit=10" \
  -H "Authorization: Bearer <token>"
```
