title: "Update KYC Record"
description: "Update an existing KYC (Know Your Customer) record by token ID."
---

# PATCH `/api/kyc/{tokenId}`

Update an existing KYC record.

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `tokenId` | `string` | yes | KYC token ID (path parameter) |

## Request Body

> Content-Type: `application/json`

```json
{
  "tokenId": "kyc-uuid-123",
  "owner": "0x123abc...",
  "fileHash": "Qmabcdef...",
  "metadataUrl": "https://ipfs.io/ipfs/Qmabcdef...",
  "documentUrl": "https://storage.example.com/docs/kyc-123.pdf",
  "name": "NFT-123",
  "description": "KYC for user 0x123abc...",
  "txHash": "0xabc123...",
  "createdAt": 1699286400000,
  "updatedAt": 1699372800000,
  "status": "Pending"
}
```

## Responses

### 200 — OK

KYC record successfully updated.

```json
{
  "tokenId": "kyc-uuid-123",
  "owner": "0x123abc...",
  "fileHash": "Qmabcdef...",
  "metadataUrl": "https://ipfs.io/ipfs/Qmabcdef...",
  "documentUrl": "https://storage.example.com/docs/kyc-123.pdf",
  "name": "NFT-123",
  "description": "KYC for user 0x123abc...",
  "txHash": "0xabc123...",
  "createdAt": 1699286400000,
  "updatedAt": 1699372800000,
  "status": "Pending"
}
```

### 400 — Bad Request

Returned when the request body is invalid or missing required fields.

```json
{
  "error": "Invalid KYC update data"
}
```

### 401 — Unauthorized

Returned if the user is not authenticated.

```json
{
  "error": "Unauthorized"
}
```

## Authentication

This endpoint requires a **Bearer Token** for authentication.

## Example Request

```bash
curl -X PATCH "https://api.zephyra.dev/api/kyc/kyc-uuid-123" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "owner": "0x123abc...",
    "fileHash": "Qmabcdef...",
    "metadataUrl": "https://ipfs.io/ipfs/Qmabcdef...",
    "documentUrl": "https://storage.example.com/docs/kyc-123.pdf",
    "name": "NFT-123",
    "description": "KYC for user 0x123abc...",
    "txHash": "0xabc123...",
    "status": "Pending"
  }'
```
