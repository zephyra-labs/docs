# POST `/api/trade/{id}/participant`

Add a participant to a trade.

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `id` | `string` | yes | Trade ID (path parameter) |

## Request Body

> Content-Type: `application/json`

```json
{
  "address": "0xabc123...",
  "role": "exporter",
  "kycVerified": false,
  "walletConnected": true
}
```

## Responses

### 200 — OK

Returns the updated trade record including the new participant.

```json
{
  "id": "trade-uuid-123",
  "contractAddress": "0xcontract...",
  "participants": [
    {
      "address": "0xabc123...",
      "role": "exporter",
      "kycVerified": false,
      "walletConnected": true
    }
  ],
  "status": "draft",
  "currentStage": 1,
  "createdAt": 1699286400000,
  "updatedAt": 1699372800000
}
```

### 400 — Bad Request

Returned if the request body fails validation.

```json
{
  "error": "Validation error"
}
```

### 401 — Unauthorized

Returned if the user is not authenticated.

```json
{
  "error": "Unauthorized"
}
```

## Authentication

This endpoint requires a **Bearer Token** for authentication.

## Example Request

```bash
curl -X POST "https://api.zephyra.dev/api/trade/trade-uuid-123/participant" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "address": "0xabc123...",
    "role": "exporter",
    "kycVerified": false,
    "walletConnected": true
  }'
```
