# GET `/api/dashboard/admin`

Fetch system-wide metrics and recent activities for the admin dashboard.  
Accessible by admin users only.

## Authentication

This endpoint requires a **Bearer Token** with admin privileges.

## Example Request

```bash
curl -X GET "https://api.zephyra.dev/api/dashboard/admin" \
  -H "Authorization: Bearer <admin-token>"
```
