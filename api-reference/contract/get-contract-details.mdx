# GET `/api/contract/{address}/details`

Fetch detailed information for a specific contract, including its current state and history.

## Path Parameters

| Name      | Type     | Required | Description        |
|-----------|----------|----------|------------------|
| `address` | `string` | yes      | Contract address  |

## Authentication

This endpoint requires a **Bearer Token**.

## Example Request

```bash
curl -X GET "https://api.zephyra.dev/api/contract/0xABC123/details" \
  -H "Authorization: Bearer <token>"
```
