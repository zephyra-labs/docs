# GET `/api/contract`

Fetch all deployed contracts in the system along with their current state and history.

## Query Parameters

This endpoint does not accept any query parameters.

## Authentication

This endpoint requires a **Bearer Token**.

## Example Request

```bash
curl -X GET "https://api.zephyra.dev/api/contract" \
  -H "Authorization: Bearer <token>"
```
