## API Documentation with Swagger

Zephyra uses **Swagger / OpenAPI 3.0** to generate interactive API documentation for developers.  
This allows you to explore endpoints, request/response schemas, and try out APIs directly from the browser.

---

## Setup Swagger

<Steps>

Run the following in the backend folder:

```bash
pnpm install swagger-ui-express yamljs
```

Create a file `swagger.yaml` in `src/docs/`:

```yaml
openapi: 3.0.3
info:
  title: Zephyra API
  version: 1.0.0
  description: API documentation for Zephyra backend
servers:
  - url: http://localhost:5000/api
paths:
  /users:
    get:
      summary: List all users
      responses:
        '200':
          description: OK
```

Add Swagger middleware to your backend server:

```ts
// src/app.ts or src/server.ts
import swaggerUi from "swagger-ui-express";
import YAML from "yamljs";
import express from "express";

const app = express();
const swaggerDocument = YAML.load("./src/docs/swagger.yaml");

app.use("/api/docs", swaggerUi.serve, swaggerUi.setup(swaggerDocument));

app.listen(process.env.PORT || 5000, () => {
  console.log(`Server running on http://localhost:${process.env.PORT || 5000}`);
  console.log(`Swagger docs available at http://localhost:${process.env.PORT || 5000}/api/docs`);
});
```

Open your browser and go to:

```
http://localhost:5000/api/docs
```

You can now interact with all documented API endpoints.

</Steps>
