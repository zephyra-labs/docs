## Zephyra CLI

Zephyra provides several **CLI commands** to streamline development, testing, and deployment.  
All commands are executed using **pnpm** for consistency and speed.

---

## Common Commands

<Steps>

Run development server
```bash
pnpm dev
```
Build for production
```bash
pnpm build
```
Start production server
```bash
pnpm start
```
Lint and format code
```bash
pnpm lint
pnpm format
```
Run tests
```bash
pnpm test
```
Generate documentation
If you’re using **Swagger** for API documentation:
```bash
pnpm swagger:generate
```
</Steps>

## CLI Environment

Ensure that your `.env` file is loaded before running any CLI command.
If you’re using multiple environments (e.g., `staging`, `production`), you can specify it:

```bash
NODE_ENV=staging pnpm start
```
